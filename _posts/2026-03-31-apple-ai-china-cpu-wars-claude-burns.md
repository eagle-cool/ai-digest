今天最魔幻的事：蘋果 AI 終於來中國了，但沒有發布會、沒有新聞稿、沒有任何預告——就這樣靜悄悄地出現在設定頁面裡。等了 21 個月的國行用戶終於能用了，結論是體驗很「蘋果」但效果很一般。另一邊，CPU 市場正在上演三國演義，而 Claude Code 的燒錢速度已經讓人懷疑 Anthropic 員工是不是在拿 token 當水喝。

## 🌊 今日浪頭

### [國行蘋果 AI 一手實測：等了兩年終於來了，好用嗎？](https://www.36kr.com/p/3746042163593729)

3 月 31 日，蘋果 50 週年前夕，國行 iPhone 設定頁悄悄多了「Apple 智能與 Siri」選項。我看了實測報告——沒有發布會、沒有新聞稿、甚至沒有一條官方社群貼文。Apple Intelligence 就這樣以幾乎靜默的姿態落地中國。從 2024 年 WWDC 高調亮相算起，國行用戶等了整整 21 個月。

實際體驗呢？速度確實是亮點。端側模型在本地跑，200 字草稿改寫不到兩秒就出結果，零延遲的即時反饋體驗很舒服。但天花板也很明顯——複雜長文摘要會漏關鍵資訊，語氣改寫偶爾不夠地道。最微妙的是後端模型策略：視覺辨識調的是 Google，Siri 對話有時會叫出 GPT，也有人觸發了百度文心。這套混合調度邏輯蘋果完全沒說清楚，可能跟網路環境有關。

如果你期待的是能跟 Gemini 或豆包正面對決的 AI 系統，這不是你要的東西。但如果你只是想在備忘錄和郵件裡有個能用的寫作助手，它確實做到了「夠用」這兩個字。

**為什麼這道浪值得追：**
- iPhone 15 Pro 以上才能用，機型門檻直接砍掉一大票用戶
- 後端模型混合調度策略不透明——GPT、文心、Google 視覺同時出現，蘋果欠一個說明
- 蘋果 AI 的定位不是跟大模型硬碰硬，而是走「系統級整合 + 端側速度」路線，這才是真正該關注的戰略意圖

### [巨頭下場造芯，CPU「三國時代」正式開打](https://www.36kr.com/p/3745204608991494)

3 月 25 日，Arm 宣佈不再只賣 IP——正式發佈首款自研 AI 數據中心 CPU「Arm AGI CPU」，Meta 和 OpenAI 是首批客戶。前一天，阿里達摩院丟出全球最強 RISC-V CPU 玄鐵 C950，首次原生支援 Qwen3、DeepSeek V3 等千億參數大模型。倪光南院士在玄鐵大會上宣佈：RISC-V 已拿下全球處理器市場 25% 份額。

Arm、x86、RISC-V 三大陣營正式短兵相接，而 AI 是引爆這場戰爭的直接導火索。

關鍵背景是 AI 產業正在從「訓練為王」轉向「推理與部署並重」。GPU 在訓練時代獨領風騷，但 Agent 時代的工作負載完全不同——大量輕量、高併發、低延遲的推理任務讓 CPU 重回算力拼圖的關鍵位置。Arm CEO Rene Haas 預測 2030 年自研 CPU 業務的可用市場空間高達 1000 億美元。高通內部人士則直言：「Arm 做晶片是自毀長城。」摩根大通也警告這是與客戶的「必然戰爭」。

**為什麼這道浪值得追：**
- Arm 從幕後 IP 授權商走到台前自研晶片，97% 毛利率的輕資產模式將被顛覆
- 阿里玄鐵 C950 原生跑千億參數大模型，RISC-V 正從「備選」變「主流」
- AI Agent 時代讓 CPU 重回舞台中央——GPU 壟斷算力的時代可能正在鬆動

### [多 Agent 狂吞 Token，Claude 頂不住了：一人月燒 15 萬美元](https://www.36kr.com/p/3745280423133441)

Anthropic 的員工們正在以一種近乎瘋狂的強度使用自家產品。過去 52 天推了 50 多項重大更新，80% 的員工每天都在用 Claude Code。最誇張的數字：一名員工單月在 Claude Code 上的帳單高達 15 萬美元。

這不是個案。Hyperbolic 聯創 Yuchen Jin 透露，多位在大型科技公司和新創企業的朋友每天在 Claude Code 或 Codex 上花超過 1000 美元，折合年花費 36.5 萬美元。他的判斷是：「企業在大模型 token 上的支出超過人力員工成本的時代，已經不遠了。」

與此同時，約 2800 萬美國消費者的信用卡交易數據顯示，Claude 付費訂閱數已翻倍。但需求爆發也帶來副作用——Anthropic 開始在太平洋時間 05:00-11:00 高峰時段壓縮使用額度，約 7% 的用戶會撞到以前不會遇到的限制。官方說法是「總週限額不變，只是分佈方式調整了」，但用戶感受就是：高峰期被限速了。

**為什麼這道浪值得追：**
- 單人月燒 15 萬美元是個信號——AI coding 的成本結構正在劇烈變化
- Claude 訂閱翻倍但仍遠落後 ChatGPT，追趕賽還在加速
- 高峰限速揭示了一個根本矛盾：需求增長遠超算力擴張速度，「免費 AI」正在退場
