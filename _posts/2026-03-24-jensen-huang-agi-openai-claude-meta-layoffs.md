## 🌊 今日浪頭

### [Nvidia CEO Jensen Huang says 'I think we've achieved AGI'](https://www.theverge.com/ai-artificial-intelligence/899086/jensen-huang-nvidia-agi)

老黃在 Lex Fridman 的播客上丟了一句讓全網炸鍋的話：「I think we've achieved AGI.」我看了完整片段，Fridman 定義的 AGI 是「能獨立創辦並經營一家十億美元公司的 AI」，老黃直接回了「I think it's now」。不過有趣的是，他接著又自己踩了剎車——「十萬個 Agent 要造出 NVIDIA？機率是零。」這就是老黃的風格，先把天花板掀了，再輕描淡寫補一句。

**為什麼這道浪值得追：**
- AGI 這個詞最近被各大 CEO 玩壞了，老黃的發言等於是直接把辯論推到新高度——而且他不是模型廠商，是賣鏟子的人在喊
- 他提到 OpenClaw 的病毒式成功，暗示 AI Agent 已經是 AGI 的早期形態
- 但「走回頭路」的補充也很關鍵——目前的 Agent 能做很多事，但離真正取代人類組織運作還有巨大鴻溝

### [OpenAI 急招 3500 人反擊，Claude 搶走 70% 新單](https://www.36kr.com/p/3735314067406853)

這篇我看完整個人都精神了。根據支付平台 Ramp 的數據，首次採購 AI 的企業選擇 Anthropic 的比例是 OpenAI 的三倍；70% 的新企業 AI 合約流向了 Anthropic。Claude 訂閱量上月漲了 4.9%，OpenAI 反而跌了 1.5%。奧特曼年初才說要「大幅放緩招聘」，轉眼就宣布今年要招 3500 人把團隊翻倍——其中還包括一個新職位叫「技術大使」，專門派去企業幫忙落地。說白了，這是 Anthropic 早就在做的事。

**為什麼這道浪值得追：**
- Stargate 計畫已實質終止，OpenAI 放棄自建資料中心，改從微軟、甲骨文和 AWS 租算力——從 1.3 兆美元砍到 6650 億
- Sora 用戶新鮮感一過就涼了，ChatGPT 廣告沒拿到廣告主信任，Codex 還在追 Claude Code
- 有投資人警告 OpenAI 恐陷入「no man's land」——Google 搶聊天用戶、Anthropic 攻企業市場，OpenAI 兩頭落空

### [Meta 裁 1.6 萬人省錢，轉手砸 6000 億搞 AI](https://www.36kr.com/p/3735378858574088)

扎克伯格的算術題很殘酷但也很清楚：砍掉 20% 的人（約 1.6 萬人），把錢全投到 AI。6000 億美元建資料中心、143 億挖 Scale AI 創辦人 Alexandr Wang、20 億買下 Manus。更諷刺的是，Meta 新一代模型 Avocado 性能還追不上 Gemini 3 和 Claude——內部甚至討論過暫時租 Google 的 Gemini 來頂。花了幾千億搞 AI，結果可能要借競爭對手的模型過渡，這畫面真的有點魔幻。

**為什麼這道浪值得追：**
- 這不是 2022 年那種「疫情膨脹後的瘦身」，而是結構性重組——AI 讓很多崗位變得「多餘」
- Block 裁了近一半員工、Amazon 也砍了 1.6 萬人，硅谷的「AI 換人」多米諾骨牌正式倒下
- Yann LeCun 離開 Meta 去創業、Llama 4 翻車、Behemoth 被砍——Meta 可能正在從開源轉向閉源策略
