## 🌊 今日浪頭

### [Lovable says it added $100M in revenue last month alone, with just 146 employees](https://techcrunch.com/2026/03/11/lovable-says-it-added-100m-in-revenue-last-month-alone-with-just-146-employees/)

146 個人，年收 $400M。我反覆看了三遍數字，沒有看錯。

Lovable 在二月正式突破 $400M ARR，比一月的 $300M 又多了一億美元——也就是說，**一個月淨增 $100M 營收**。更誇張的是，這是靠 146 名全職員工達成的，每人平均創造 $2.77M 的年營收。Gartner 預測 2030 年會出現一批「每員工 $2M ARR」的新一代獨角獸，Lovable 已經在 2026 年就超越了這個數字。

超過半數 Fortune 500 在用 Lovable，客戶包括 Klarna 和 HubSpot。國際婦女節當天全平台免費開放，單日建立或更新的專案數超過 50 萬個，是平常日均的 2.5 倍。CEO Anton Osika 面對 Claude Code 和 Codex 的競爭壓力，表現得非常淡定——看到這個增長曲線，他確實有淡定的本錢。

**為什麼這道浪值得追：**
- 每員工 $2.77M ARR 是軟體業見過的最瘋狂效率，連 AI 時代之前的 SaaS 最佳紀錄都被碾壓
- Vibe Coding 不再只是 indie hacker 的玩具——Fortune 500 也在用，企業市場才是真正的金礦
- 從 $100M → $200M → $300M → $400M，增速不是在放緩而是在加速——這才是最可怕的

### [Replit snags $9B valuation 6 months after hitting $3B](https://techcrunch.com/2026/03/11/replit-snags-9b-valuation-6-months-after-hitting-3b/)

半年，估值三倍跳。九年磨一劍，一朝 Vibe Coding。

Replit 宣布完成 $400M D 輪融資，估值從去年九月的 $3B 直接跳到 $9B。領投是老投資人 Georgian Partners，跟投陣容豪華到不行：a16z、Coatue、Y Combinator、Databricks Ventures——連 Shaquille O'Neal 和 Jared Leto 都進來了（是的，你沒看錯）。CEO Amjad Masad 說目標是年底前 ARR 衝到 $1B。

但讓我最有感的其實是 Masad 之前說的那句話：Replit 花了九年才找到市場。從服務專業開發者到轉向非程序員，這個爭議性的 pivot 差點讓團隊分裂，結果反而成了引爆點。Vibe Coding 的浪潮簡直像是為 Replit 量身打造的——他們已經在那裡等了九年。

**為什麼這道浪值得追：**
- 六個月估值翻三倍，速度連 AI 時代都罕見——市場對 Vibe Coding 的信心正在指數級增長
- 去年九月 $150M ARR，現在目標 $1B ARR，增長曲線幾乎是垂直的
- 九年磨劍的故事也提醒我們：有時候不是你的方向不對，是時代還沒到

### [Nvidia is reportedly planning its own open source OpenClaw competitor](https://arstechnica.com/ai/2026/03/nvidia-is-reportedly-planning-its-own-open-source-openclaw-competitor/)

當黃仁勳說 OpenClaw 是「有史以來最重要的軟體發布」的時候，原來他不只是在讚美——他是在宣戰。

根據 Wired 報導，Nvidia 正在準備推出自己的開源 AI Agent 平台「NemoClaw」，已經在跟 Salesforce、Cisco、Google、Adobe、CrowdStrike 等企業巨頭洽談合作。重點來了：NemoClaw 主打**企業級安全和隱私工具**——直接對準了 OpenClaw 最大的軟肋。

NemoClaw 可以在沒有 Nvidia GPU 的機器上運行——嘴上說不介意，但 Nvidia 的如意算盤很清楚：Agent 跑起來是要燒 token 的，token 背後是推理算力，推理算力背後是 GPU。讓更多企業跑 Agent → 消耗更多算力 → 買更多 Nvidia 的卡。這招「賣水給淘金者」的生意，黃仁勳玩得太溜了。更耐人尋味的是時間點——下週就是 GTC 開發者大會，NemoClaw 很可能就是壓軸大菜。

**為什麼這道浪值得追：**
- Nvidia 從晶片公司正式跨入 AI Agent 軟體平台，這是產業格局的根本性擴張
- 企業級安全是 OpenClaw 生態的最大痛點，NemoClaw 如果做好這塊，企業市場可能直接被收割
- 「免費平台 + 付費算力」的商業模式——但 Nvidia 掌握的是比雲端更底層的資源
